# A Page for Ellen Moran's Music

To add a new song to the page, upload the mp3 file to the public/music folder. Ensure the file has the correct attributes as that will be displayed on the player.

They are displayed in alphabetical order. The filename isn't published directly so can be used to order the tracks e.g. 01-first-track-name.mp3 , 02-second-track-name.mp3 and so on.
